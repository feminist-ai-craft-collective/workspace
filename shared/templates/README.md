# Shared Templates

Starter files and patterns anyone can copy and customize.

---

## What's Here

Templates for common things:
- Project folder structures
- CLAUDE.md starters
- Documentation patterns
- Workflow templates

---

## How to Use

1. Find a template that fits your need
2. Copy it to your project folder
3. Customize it

```bash
cp -r shared/templates/template-name projects/your-name/
```

---

## How to Contribute

Created something reusable? 
1. Generalize it (remove personal details)
2. Add clear comments or instructions
3. Post in Discussions
4. Add it here
