# Shared Contexts

Collective prompts, personas, and system instructions that anyone can use or build on.

---

## How to Use

1. Browse the files here for inspiration
2. Copy what you need to your own project folder
3. Customize for your purposes

Or reference directly in your CLAUDE.md:
```markdown
See `../../shared/contexts/filename.md` for base context.
```

---

## How to Contribute

1. Create and test in your own project folder first
2. When it's useful and generalizable, post in Discussions
3. If others want it, add it here with a descriptive name

---

## Naming Convention

Use descriptive, lowercase, hyphenated names:
- `thoughtful-educator.md`
- `structured-brainstorm.md`
- `gentle-feedback.md`
