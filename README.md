# Feminist AI Craft Collective

**Learn by doing and do better.**

A shared space for AI enthusiasts collaborating on Claude Cowork projects — contexts, skills, environments, and experiments.

---

## What's Here

```
├── shared/              # Collective resources everyone can use
│   ├── contexts/        # System prompts, personas, shared instructions
│   ├── skills/          # Reusable SKILL.md files
│   └── templates/       # Starter files and patterns
│
├── projects/            # Individual member workspaces
│   └── [your-name]/     # Your personal project space
│
├── docs/                # Guides and how-tos
│
└── ops/                 # Operational checklists and safety guides
```

---

## Getting Started

1. **Clone the repo** to your local machine
2. **Create your project folder** under `projects/` (copy an existing one or start fresh)
3. **Open Claude Code** in your project folder — it will read your `CLAUDE.md` for context
4. **Start experimenting!**

See [docs/getting-started.md](docs/getting-started.md) for the full walkthrough.

---

## How We Work

- **Your folder, your space** — work freely in `projects/[your-name]/`
- **Shared resources** — post in Discussions before editing `shared/`
- **No secrets in the repo** — check `ops/secrets-check.md` before committing
- **Pull before you push** — keeps everyone in sync
